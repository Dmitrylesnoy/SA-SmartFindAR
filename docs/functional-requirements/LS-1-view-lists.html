📋 LS-1: Просмотр списка доступных вещей

Основная информация​

СвойствоЗначение
IDLS-1
НазваниеПросмотр списка доступных вещей

Акторы​

  • App
  • Пользователь
  • Сервер

Предусловия​

  1. У пользователя есть интернет подключение

Основной поток​

  1. Пользователь открывает App и видит окно списка доступных вещей

  2. App отправляет запрос синхронизации на Сервер

  3. Сервер синхронизирует вещи общего доступа пользователя

  4. Сервер отправляет App ответ — данные доступных вещей из хранилища пользователя

  5. App отображает полученные данные вещей в списке доступных вещей

Постусловия​

Успех​

  1. Данные о доступных пользователю вещах выгружены из хранилища сервера на App
  2. Пользователь видит список доступных вещей и их информацию в карточках

Неудача​

  1. Список доступных вещей не обновлён, взаимодействие с карточками вещей заблокировано. Пользователь видит сообщение "Ошибка синхронизации"

Альтернативные потоки​

У пользователя нет сохранённых вещей​

Точка входа: Шаг 2

2.1. Сервер отвечает сообщением об отсутствии сохранений вещей в хранилище пользователя

2.2. App показывает пользователю сообщение, что у него нет сохранённых вещей. Список вещей пуст

Исключительные потоки​

Сервер недоступен​

Точка входа: шаг 6

6.1. App не получает ответа от сервера

6.2. App выводит пользователю сообщение: "Сервер временно недоступен, попробуйте позже"

6.3. App переходит на окно просмотра списка предметов
