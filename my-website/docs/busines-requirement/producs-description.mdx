# Описание продукта

---

**Общая концепция**:

- Интеллектуальная система позволяет точно найти сохранённую вещь и указать где она лежит, опираясь на 3д сканирование помещения, компьютерное зрение и машинное обучение

---

**Ключевые функции**:

- Определение местоположения предмета

- Построение маршрута в пространстве к предмету

- Сохранение всех предметов и их локаций

---

:::info
Продукт ориентирован на использование в помещениях с возможностью масштабирования на различные типы пространств.
:::
