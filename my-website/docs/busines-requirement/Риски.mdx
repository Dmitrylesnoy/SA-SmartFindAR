**6. Риски**

**Риски и стратегии управления рисками**:

- Низкая точность позиционирования местоположения пользователя

  - Описание: Как следствие — низкая точность алгоритмов навигации, Ухудшение UX, потеря аудитории

  - **Вероятность**: средняя. **Важность**: высокая

  - **Решение**: использование сенсоров смартфона - Wi-Fi, Bluetooth, GPS

- Утечка конфиденциальности данных

  - Описание: Потеря доверия пользователей, аудитории. Угроза безопасности и целостности системы

  - **Вероятность**: средняя. **Важность**: высокая

  - Решение: локальная обработка данных, прозрачная политика конфиденциальности, шифрование, инфраструктура инф. безопасности

- Ограничения аппаратного обеспечения клиента:

  - Описание: Производительность приложения может быть низкой на устройствах клиента

  - **Вероятность:** средняя. **Важность:** средняя

  - Решение: поддержка минимальных требований к устройствам, оптимизации приложения

- Отсутствие необходимых средств на смартфоне пользователя

  - Описание: На смартфоне пользователя могут отсутствовать необходимые для работы приложения датчики и модули (AR, GPS)

  - **Вероятность:** низкая, **Важность:** высокая

  - Решение: проверка устройства пользователя на удовлетворение требований установки приложения

- Конкуренция

  - Описание: борьба за аудиторию с конкурентами / потеря аудитории

  - **Вероятность:** высокая. **Важность**: высокая

  - Решение: анализ рынка, улучшение UX, удовлетворение потребностей пользователя. Добавление новых фич

- Исчерпание бюджета

  - Вероятность: низкая, Важность: средняя

  - Описание: непредвиденные затраты на реализацию, нехватка бюджета до выпуска продукта

  - Решение: Запрос дополнительных средств финансирования, презентация перед инвесторами
