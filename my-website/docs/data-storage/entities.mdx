# Сущности

|**Сущность**|**Атрибуты (Поля)**|**Акторы и характер взаимодействия**|**Тип данных**|**Правила и ограничения**|
|-|-|-|-|-|
|**Пользователь (User)**|ID (int)<br/>Name (string)<br/>Email (string) <br/>Phone (string)<br/>PasswordHash (text)|**Приложение:** создание, чтение, аутентификация.|Транзакционные|Обязательно наличие Email + Password И/ИЛИ Phone. <br/>Email уникален.|
|**Помещение (Premises)**|ID (int)<br/>Owner_ID (int**)**<br/>Name (string)<br/>Description (text)<br/>Model3D (object)|**Приложение:** создание/ред. помещения. <br/>**Алгоритмы распознавания:** чтение 3D-модели.|Транзакционные|Owner_ID должен ссылаться на существующего User. <br/>Model3D - обязательный файл.|
|**Предмет (Item)**|ID (int)<br/>Owner_ID **(**int**)**<br/>Name (string)<br/>Description (text)|**Приложение:** создание, изменение, поиск.|Транзакционные|Name, Owner_id не может быть пустым. <br/>Предмет принадлежит существующему User.|
|**Медиа предмета (Item_media)**|ID (int)<br/>Item_ID (int)<br/>FilePath (string)<br/>Hash (string)|**Приложение:** загрузка эталонных фото. <br/>**Алгоритмы распознавания:** чтение для распознавания.|Транзакционные|Связь с Item (1:M). <br/>Уникальность пары (Item_ID, Hash) для защиты от дублей.|
|**Точка (Point)**|ID (int)<br/>Premises_ID (int)<br/>X (float)<br/>Y (float)<br/>Z (float)|**Алгоритмы распознавания:** запись координат. <br/>**Приложение:** чтение для визуализации.|Транзакционные|Координаты привязаны к конкретной Model3D помещения.|
|**Местоположение (Location)**|ID (int)<br/>Item_ID (int)<br/>Point_ID (int)<br/>Photo_Ref (string)<br/>Timestamp (datetime)|**Алгоритмы распознавания:** создание записи. <br/>**Приложение:** чтение истории и текущего места.|Транзакционные (алгоритмы)<br/>Аналитические (приложение)|При каждом новом распознавании создается новая запись с актуальным Timestamp.|
|Нейросеть по 3д модели (weights)|ID: (int)<br/>Premises_ID (int)<br/>data (object)<br/>accuracy (float)<br/>last_update (timestamp)|**Алгоритмы распознавания:** создание и обновление записи набора весов адаптированной под 3д модель. Чтение весов<br/>|Транзакционные|data не может быть пустым|
