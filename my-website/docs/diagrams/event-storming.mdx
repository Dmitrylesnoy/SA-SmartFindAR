## Описание

Event Storming используется для исследования предметной области и выявления:

- **События** (Events) — факты, которые произошли в системе
- **Команды** (Commands) — действия пользователей или системы
- **Агрегаты** (Aggregates) — группы связанных объектов
- **Политики** (Policies) — правила реагирования на события

На диаграмме представлен процесс просмотра местоположения предмета на 3D модели помещения.
