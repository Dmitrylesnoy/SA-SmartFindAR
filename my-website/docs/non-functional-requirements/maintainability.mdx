# Поддерживаемость

## Независимость компонентов

| Свойство | Значение |
|:---------|:----------|
| **ID** | MAN-01 |
| **Название** | Независимость компонентов |
| **Приоритет** | Высокий |

**Описание:** ML-модели должны быть выделены в отдельные сервисы с четким API, чтобы их можно было обновлять и дообучать независимо от основного приложения.

**Обоснование:** Модели распознавания будут улучшаться. Возможность выкатить новую модель без пересборки всего приложения экономит ресурсы.

## Документация API

| Свойство | Значение |
|:---------|:----------|
| **ID** | MAN-02 |
| **Название** | Документация API |
| **Приоритет** | Средний |

**Описание:** Все методы взаимодействия клиента и сервера должны быть задокументированы (Swagger/OpenAPI).

**Обоснование:** В команде бэкенд и Android-разработчик. Документация — их единственный способ синхронизации.
