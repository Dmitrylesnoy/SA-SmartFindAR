# Безопасность

## Зашифрованное хранение данных

| Свойство | Значение |
|:---------|:----------|
| **ID** | SEC-01 |
| **Название** | Зашифрованное хранение данных |
| **Приоритет** | Высокий |

**Описание:** 3D-модели помещений и фотографии предметов должны храниться в зашифрованном виде в хранилище пользователя на сервере и локально на устройстве. На сервер передаются, в том числе обезличенные метаданные для аналитики.

**Обоснование:** Если сервер взломают, злоумышленники не получат фото вашего дома, вещей и документов.

## Защита общего доступа

| Свойство | Значение |
|:---------|:----------|
| **ID** | SEC-02 |
| **Название** | Защита общего доступа |
| **Приоритет** | Средний |

**Описание:** Доступ к общему каталогу предметов должен требовать подтверждения (биометрия устройства или пин-код) при попытке просмотра или редактирования.

**Обоснование:** Если телефон потеряет член семьи, посторонний не увидит, где лежат ключи или документы.
