# Платформизация

### Модель бизнеса: B2B2C и B2B

Основной моделью является **B2B2C, та**кже возможна модель **B2B** для внутреннего использования компаниями.

### Целевые сегменты:

- **Ритейл мебели и товаров для дома:** Компании уровня **IKEA, Hoff** или **Леруа Мерлен**

  - Платформа (сайт или приложение) для взаимодействия с пользователем - AR показ предметов, чтобы посмотреть их в интерьере своей квартиры

  - Магазину предоставляется инструмент 3д сканирования мебели, для дальнейшего отображения её в AR у пользователя. Система предобрабатывает 3д модели предметов и мебели для оптимизации отображения на мобильном устройcтве. 

  - В приложении\сайте предоставляется список мебели (товаров в магазине) и их расположение в помещении магазина, чтобы при помощи приложения клиент смог легко дойти в нужное место.

  - Клиент может сделать 3д модель своей квартиры и далее, в магазине, отсканировав лейбл понравившегося предмета - смоделировать его расположение квартире.

- **Страховые компании:** Для создания точного цифрового реестра имущества клиента, что критично при расчете страховых премий и верификации страховых случаев.

- **Небольшие склады**, где внедрение RFID слишком дорого, но нужно быстро находить конкретные коробки с помощью смартфона

### Стратегия монетизации

Для обеспечения масштабируемости и стабильного денежного потока выбирается гибридная стратегия:

- **Периодическая подписка - основной доход:**

  - **Для B2B:** Подписка на расширенное облако для хранения более 2-х помещений и историю перемещений вещей (аналитика).

  - **Для B2B2:** Ежемесячные платежи за количество активных «точек учета» или помещений под управлением.

- **Плата за объем данных:**

  - Учитывая, что 3D-модели помещений весят много, то вводится тарификация за хранение данных сверх лимита.

- **Лицензирование:**

  - Продажа прав на использование всего стека крупным корпорациям для встраивания в их закрытые контуры.
